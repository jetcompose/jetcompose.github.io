Button in Jetpack compose

To add a button in Jetpack Compose, you can use the Button composable. The Button composable allows you to specify the text to be displayed on the button, as well as an action to be performed when the button is clicked. Here's an example of how to use the Button composable: 

                        
                                
                                    Button(onClick = {
                                        // Perform some action when the button is clicked
                                    }) {
                                        Text("Click me")
                                    }

The onClick parameter is a lambda function that specifies the action to be performed when the button is clicked. The lambda function can contain any code that you want to execute when the button is clicked.

The Text composable inside the lambda function specifies the text to be displayed on the button. You can customize the appearance of the button by setting various attributes, such as the color and fontSize.

Here's an example of a button with customized appearance:

                        
                                
                                    Button(onClick = {
                                        // Perform some action when the button is clicked
                                    }, color = Color.Blue, fontSize = 16.sp) {
                                        Text("Click me")
                                    }

You can also customize the appearance of the button by using the optional modifier parameter. For example, you can use the background modifier to set the background color of the button: 

                        
                                
                                    Button(
                                            onClick = {
                                                // Add your code here to handle the click event
                                            },
                                            text = {
                                                Text("Button text")
                                            },
                                            modifier = Modifier.background(color = Color.Red)
                                        )

This will create a red button with the text "Button text".
